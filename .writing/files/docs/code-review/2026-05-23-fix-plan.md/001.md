#### #8 消除索引状态保存重复代码

三处重复逻辑位置：
- `tools.go:163-178`（全量构建后保存）
- `index_manager.go:106-118`（fullBuild 内）
- `index_manager.go:279-291`（incrementalUpdate 内）

**方案**：在 `IndexStateStore` 上添加便捷方法：

```go
// internal/indexer/index_state.go 中添加：
func (s *IndexStateStore) SaveFromStats(projectPath string, stats types.IndexStats, extensions []string) {
    fingerprint, mtimes, _ := s.GetCurrentFingerprint(projectPath, extensions)
    state := &types.IndexState{
        LastIndexedAt: time.Now(),
        Fingerprint:   fingerprint,
        TotalFiles:    stats.TotalFiles,
        TotalChunks:   stats.TotalChunks,
        ProjectPath:   projectPath,
        FileMtimes:    mtimes,
    }
    if err := s.Save(state); err != nil {
        slog.Warn("保存索引状态失败", "err", err)
    }
}
```

三处调用点替换为 `stateStore.SaveFromStats(projectPath, stats, cfg.ScanExtensions)`。

**注意**：`incrementalUpdate` 中的 `TotalChunks` 计算方式略有不同（`state.TotalChunks - len(changedFiles) + chunkIdx`），可作为可选参数或在调用前计算好传入 stats。

#### #5 去掉 GetEmbedding 隐式单例

**文件**：`internal/embedding/embedding.go`

采用方案 A：删除 `sync.Once` 全局变量，`GetEmbedding` 每次调用 `NewEmbeddingProvider(cfg)` 创建实例。

```go
// 删除：
var (
    sharedProvider     EmbeddingProvider
    sharedProviderOnce sync.Once
    sharedProviderErr  error
)

// GetEmbedding 修改为：
func GetEmbedding(cfg config.Config, text string) ([]float32, error) {
    provider, err := NewEmbeddingProvider(cfg)
    if err != nil {
        return nil, err
    }
    return provider.Embed(context.Background(), text)
}
```

**前提验证**：确认 `NewEmbeddingProvider` 仅做结构体初始化（不建立网络连接），HTTP 连接在 `Embed` 调用时才建立，以确保此方案无性能回退。
